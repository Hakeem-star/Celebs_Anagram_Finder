# Celebs_Anagram_Finder
This allows you to find the Celebrity in an anagram (if they are in the database) e.g.TRUMFDAONDL

Codepen available at - https://codepen.io/Hakeem-star/pen/qBWZzab

This celebrity anagram finder accepts single anagrams or a comma seperated list of anagrams. 

So you can input something like TRUMFDAONDL or TRUMCDONALD,SHAVERSINB,SHAVERSINC,SHAVERSINM,SHAVERSINP
